¿Que es HTML y cual es su funcion principal en el desarrollo web?

HTML es el lenguaje que se usa para crear páginas web. Su función principal es organizar y mostrar el contenido, como texto, imágenes y enlaces, en un navegador. Es como el "esqueleto" de una página web.

¿Que son las etiquetas HTML y como se utilizan?

Las etiquetas HTML son códigos que se usan para crear y organizar el contenido de una página web, van entre símbolos Se escriben en un archivo de texto y el navegador las lee para mostrar la página correctamente.

Ejemplos de Etiquetas HTML

P: para parrafos, img: para imagenes, estos se escriben entre simbolos

¿Que es CSS y como se utiliza para dar estilo a una pagina web?

CSS es lo que hace que una página web se vea bonita. Controla colores, fuentes, tamaños y diseños. Se usa escribiendo reglas que cambian cómo se ven las etiquetas de HTML.

¿Cuales son las propiedades mas importantes de CSS?

1. color: Cambia el color del texto. 2. font-size: Ajusta el tamaño de la letra. 3. background-color: Pone color de fondo. 4. margin: Añade espacio fuera de un elemento. 5. padding: Añade espacio dentro de un elemento. 6. border: Define el borde de un elemento. 7. text-align: Alinea el texto (izquierda, centro, derecha). 8. font-family: Cambia el tipo de letra. 9. width y height: Controlan el ancho y alto. 10. display: Decide cómo se muestra un elemento.
