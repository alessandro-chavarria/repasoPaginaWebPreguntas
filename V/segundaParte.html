¿Que es el diseño responsivo y como se logra en CSS?

El diseño responsivo hace que una página web se vea bien en celulares, tablets y computadoras. En CSS, se logra con: 1. Media queries: Cambian el diseño según el tamaño de la pantalla. 2. Unidades flexibles: Usar porcentajes (%) en lugar de tamaños fijos. 3. Flexbox y Grid: Herramientas para organizar el contenido de forma adaptable.

¿Que es JavaScript y como se integra en el desarrollo web?

JavaScript hace que las páginas web sean dinámicas. Con él, puedes agregar interactividad, como mostrar mensajes o cambiar contenido. Se usa con la etiqueta script en HTML.

Explica como JavaScript interactua con el HTML y CSS a traves del DOM

JavaScript usa el DOM para: 1. Seleccionar elementos de HTML. 2. Cambiar su contenido o estilo. 3. Reaccionar a clics o eventos

Ejemplos de como JavaScript puede hacer que una pagina web sea interactiva

1. Mostrar un mensaje: javascript alert("¡Hola!"); 2. Cambiar texto: javascript elemento.innerHTML = "¡Nuevo texto!"; 3. Cambiar color: javascript body.style.backgroundColor = "azul"; 4. Validar formularios: javascript if (nombre === "") alert("¡Falta algo!"); 5. Mostrar/ocultar: javascript elemento.style.display = "none"; // Ocultar

¿Cuales son las propiedades mas importantes de CSS?

1. color: Cambia el color del texto. 2. font-size: Ajusta el tamaño de la letra. 3. background-color: Pone color de fondo. 4. margin: Añade espacio fuera de un elemento. 5. padding: Añade espacio dentro de un elemento. 6. border: Define el borde de un elemento. 7. text-align: Alinea el texto (izquierda, centro, derecha). 8. font-family: Cambia el tipo de letra. 9. width y height: Controlan el ancho y alto. 10. display: Decide cómo se muestra un elemento.
